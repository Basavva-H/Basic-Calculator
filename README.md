# Basic-Calculator
A user-friendly Python calculator capable of performing basic arithmetic operations like addition, subtraction, multiplication, and division.

# Enhanced Calculator

A modern, feature-rich desktop calculator built with Python and Tkinter, designed 
with a sleek dark UI and scientific computing capabilities.

## Features
- **Modern UI**: Dark theme with cyberpunk-inspired color palette
- **Scientific Functions**: Square root, power, and percentage calculations
- **Keyboard Support**: Full keyboard input for faster calculations
- **Interactive Design**: Smooth hover effects and responsive feedback
- **Error Handling**: Comprehensive validation for all operations

## Tech Stack
- Python 3.x
- Tkinter GUI Framework
- Math Library for scientific operations
